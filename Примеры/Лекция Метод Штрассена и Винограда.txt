Матрица

Пусть К – произвольное кольцо.

Тогда, матрицей A размера m x n над кольцом К называется прямоугольная таблица элементов кольца К, состоящая из m строк и n столбцов.

Обозначение: A = (a mxn) или

       ( a11 a12   ...  a1n  )
A = ( a21 a22   ...  a2n  )
       ( ...      ...     ... ...      )
       ( am1 am1 ... amn )

Умножение матриц

Пусть Amxn = (aij), Bmxn - (bij) - матрицы, у которых число строк B равно числу столбцов A. Тогда их произведение - матирица Сmxk = (cij), где

      n
cij = E aisbsj
     s=1 
	           ___       ___
для любого i = 1,m и j = 1,k. 

То есть,элемент матрицы Cmxk состоящий в i-строке, j-столбце, равен сумме произведений элементов i-строки матрицы А на соответствующие элементы j-столбца матрицы B.
|2|
Метод Штрассена

Пусть Amxn = (aij), Bmxn - (bij), тогда при стандартном методе произведение выражается за 8 умножений и 4 сложения. 
Но, по Лемме Штрассена данное произведение можно выразить за 7 умножений и 18 сложений.

Матрицы n x n представляются в виде блочных матриц, состоящих из четырёх подматриц размера n / 2 x n / 2 каждая. Согласно Лемме Штрассена, вычисление этого произведения сводится к умножению семи пар матриц вдвое меньшего размера, а также некоторому количеству сложений таких матриц. 
Наконец, для умножения матриц размера n 2 x n 2 рекурсивно используется этот же алгоритм.
|3|
Необходимо найти следующие значения:
d1 = (a11 + a22)(b11 + b22), d2 = (a21 + a22)b11, 
d3 = a11(b12 - b22), d4 = a22(b21 - b11)
d5 = (a11 + a12)b22, d6 = (a21 - a11)(b11 + b12)
d7 = (a12 - a22)(b21 + b22)
Затем, найти 4 подматрицы, которые составят результат:
c11 = d1 + d4 + d7 - d5, c12 = d3 + d5
c21 = d2 + d4, c22 = d1 + d3 + d6 - d2
Если умножать надо числовые матрицы размера 2 x 2, то такой способ вычисления их произведения окажется медленней, чем если считать согласно определению: действительно, легче проделать одно лишнее умножение, чем 14 лишних сложений. Смысл умножения матриц 2 x 2 за 7 умножений и много сложений в том, что эту формулу для произведения матриц можно применять рекурсивно, сводя умножение матриц большого размера (nxn) к умножению семи пар матриц вдвое меньшего размера — семи, а не восьми.

Метод Винограда

Пусть Amxn = (aij), Bnxk  = (bij). Произведение этих матриц – Cmxk = (cij). 
Тогда каждый элемент результирующей матрицы представляет собой скалярное произведение соответствующих строки и столбца. 
Пусть A = (a1,…,an), B = (b1,…,bn) - два вектора. Их скалярное произведение:
	n
A x B = E aibi, но можно упростить такую сумму 
	1
       n/2
A X B = E (a2i + b2i+1)(a2i+1 + b2i) - t
	1
        n/2     n/2
где t = E aiai+1 E bjbj+1
	1        1
|5|
Таким образом, выражения vivi+1 и wiwi+1 (где v, w – вектора) для i = 0,2,…n можно вычислить заранее для каждой соответствующей строки и столбца. Это позволяет уменьшить число умножений для больших матриц.
